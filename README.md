# Ruoyu Chen Academic Website

This is a static academic personal website for PhD applications. Open `index.html` directly in a browser to preview it.

## Current Structure

- `index.html`: main academic homepage.
- `sample-cv.html`: short web CV page. This can later be replaced with a downloadable PDF CV.
- `styles.css`: layout, typography, and color system.
- `script.js`: small header scroll enhancement.

## Content Sections

- Profile
- Education
- Research
- Publications
- Teaching
- Skills
- Contact

## Future Replacement Notes

- Replace the current text with final polished application language after you provide the full material folder.
- Add project images later only when you have selected visuals that match each research project.
- Replace `sample-cv.html` or link it to a final PDF CV before deployment.
- Add Google Scholar, GitHub, ORCID, or LinkedIn links if you want them public.

## Deployment

Because the site is static, it can be deployed on GitHub Pages, Vercel, Netlify, or any static hosting service.
